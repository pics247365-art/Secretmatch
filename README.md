# Secret Match

אפליקציית PWA למשחק התאמות זוגי.

## מה יש כאן
- אתר PWA להתקנה בטלפון
- כרטיסיות התאמה
- סימון רוצה / אולי / לא
- חישוב התאמות
- שמירה מקומית בדפדפן

## קבצים
- `index.html` — מבנה האפליקציה
- `style.css` — עיצוב
- `app.js` — לוגיקה
- `cards.json` — מאגר כרטיסים
- `manifest.json` — התקנה כ־PWA
- `service-worker.js` — עבודה אופליין
- `icon.svg` — אייקון האפליקציה
